Southeast Asia Readiness

Created a Southeast Asia readiness educational website utilizing MVC backend architecture and auth.js for user authentication, distinguishing between guest and admin access.

Upon successful login, users are directed to the dynamic homepage, featuring a slideshow-like background that transitions intervally, showcasing multiple images.

After clicking "Blog Posts", the system utilizes CRUD functionality to retrieve and display blog entries from the MSSQL database.

Users can perform CRUD operations (Create, Update, Delete) on blog posts, dynamically updating the MSSQL database in real-time.

The News Article page integrates the second CRUD functionality. In this case, the API home page was fetched from News API from newsapi.org and users can search or filter out related articles.

Unlike API homepage, the Database Homepage enables admins to create, publish, and manage custom articles, stored and retrieved directly from the MSSQL database.

Here, admin has the authority to create website by filling the blank by selecting topic, inserting headline and contents

Here, admin also can modify the news article by inserting news id and insert new details or delete the news article by simply inserting news id.
