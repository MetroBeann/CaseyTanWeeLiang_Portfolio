Auditory Reponse test how quick the elderly is able to notice the sound and click on the button

Memory Sequence test on how observant and how strong their memory are by repeating the sequence.

Developed an Advanced Clinical Frailty Scale Assessment using AI-powered motion analysis for objective frailty assessment. First, we provide user time to prepare.

After 10 seconds of preparation time, user will begin their First Movement, which is to stand straight and raise arms slowly to shoulder height, then lower them. User will repeat this as much as possible for 30 seconds.

Next, we provide user 10 seconds of rest time, so that user could prepare for Second Movement. If more time is needed, user could click on "Stop Assessment" button and start again when they are ready.

The Second Movement requires user to keep their posture straight and turn their head slowly from side to side. User will repeat this as much as possible for 30 seconds.

Another 10 seconds of rest time is provided, so that user could prepare for Third Movement. If more time is needed, user could click on "Stop Assessment" button and start again when they are ready.

The Third Movement requires user to perform a gentle walking motion in place if they feel stable. User will repeat this as much as possible for 30 seconds.

Another 10 seconds of rest time is provided, so that user could prepare for Last Movement. If more time is needed, user could click on "Stop Assessment" button and start again when they are ready.

The Last Movement requires user to perform a slow sit-to-stand motion if they feel safe or there is support nearby. User will repeat this as much as possible for 30 seconds.

Lastly, the system will accumulate all metrics from each movements, calculate the score and provide recommendations to user. However, despite webcam video processing are used to detect and analyze movement, it can't ensure if user is doing the movement correctly.
